# Canvas CLI

```bash
Usage:
	canvas new [<filename>]
	canvas list
	canvas pull <id>
	canvas delete <id>
	canvas account
	canvas login
	canvas -h | --help
	canvas --version

Options:
  -h, --help    Show this screen.
  --version     Show version.
```
