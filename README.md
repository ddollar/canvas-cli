# Canvas CLI

```bash
Usage:
	canvas new [<filename>]
	canvas list
	canvas pull <id>
	canvas delete <id>
	canvas account
	canvas login
	canvas -h | --help
	canvas --version

Options:
  -h, --help    Show this screen.
  --version     Show version.
```

## Installation

  OS X: [Canvas 0.0.1](https://s3.amazonaws.com/canvas-cli-releases/canvas-0.0.1.zip)


## Build

    ./build.sh

## Development

    go run *.go
